# Agent 学习工作台

> 重启 Obsidian，打开此 vault，从下面选一个开始。

---

## 第一步：读

打开 `concepts/` 文件夹（已配数据库视图），按顺序读：

| # | 页面 | 做什么 |
|---|------|--------|
| 1 | [[理解Agent]] | Agent 是什么、Agent Loop、核心组件 |
| 2 | [[写Skill]] | Skill 最小结构、五条铁律、实战拆解 |
| 3 | [[工具与MCP]] | 工具设计原则、MCP 标准、实战案例 |
| 4 | [[上下文工程]] | Context Window、按需注入、少即是多 |
| 5 | [[驾驭工程]] | 六层架构、工程化思维、《马书》导读 |

## 第二步：拆

打开 `projects/`，看真实源码：

| 项目 | 看什么 |
|------|--------|
| `harness-engineering-from-cc-to-ai-coding/` | 《马书》——从 Claude Code 源码学 Agent 架构 |
| `awesome-harness-engineering/` | Agent 脚手架全景图，浏览目录找感兴趣的 |
| `weread-cli/` | 小而美的 Skill+CLI 实现，代码少好读 |

## 第三步：写

打开 `skills/` 文件夹，拆解你读到的 Skill，写笔记：

- 它解决了什么问题？
- 触发条件是什么？
- 操作步骤是什么？
- 有什么坑？

## 第四步：造

挑一个你日常最重复的动作，按 `Templates/skill-notes-template` 格式，写出你第一个 Skill。
