# Coze 基础：创建第一个 Bot + 发布到微信

> 从零开始，30 分钟内做出第一个 AI Bot 并发布到微信。

---

## 一、扣子（Coze）是什么

Coze 是字节跳动的**零代码 AI 智能体开发平台**。

- **扣子** = 乐高工厂（开发平台）
- **智能体** = 乐高机器人（你做出来的 AI 助手）
- **工作流** = 机器人的控制芯片（内部处理逻辑）

**不需要编程，会打字就能做。**

---

## 二、创建第一个 Bot（15 分钟）

### Step 1：进入扣子首页

打开 https://www.coze.cn/home → 点击「创建」→「创建智能体」

### Step 2：用 AI 自动生成

1. 选择「AI 创建」
2. 输入 Bot 描述，例如：
   > "我想做一个科技知识小助手，每天给用户推送最新科技新闻，回答科技相关问题"
3. 点击「生成」，系统自动生成：
   - 人设与回复逻辑（最核心，相当于员工招聘要求）
   - 基础编排配置
   - 预览调试面板

### Step 3：测试你的 Bot

在右侧调试面板输入问题测试。不满足预期？修改人设 Prompt。

**Bot 三要素**：
```
人设与回复逻辑  → 告诉 Bot "你是谁、做什么、怎么做"
编排            → 插件 + 工作流挂载
预览与调试       → 实时测试，边改边验
```

## 三、发布到微信

### 发布渠道

| 渠道 | 说明 |
|------|------|
| 扣子商店 | 公开发布，其他人可搜索使用 |
| 豆包 | 字节自家 AI 助手平台 |
| **微信（企业）** | 需企业微信管理员授权 |
| **微信公众号（企业）** | 需认证的微信服务号 |
| 飞书 | 字节内部 IM |

### 发布步骤

1. Bot 页面右上角 → 点击「发布」
2. 勾选目标平台（扣子 / 豆包 / 飞书 / 微信）
3. 微信渠道需要：
   - 企业微信：管理员扫码授权
   - 公众号：在公众号后台绑定 Coze 开发者
4. 点击「发布」完成

⚠️ **注意**：微信个人号暂不支持直接接入，需通过企业微信或服务号绕行。

---

## 四、发布后怎么用

- **扣子商店**：生成一个链接分享，别人点开就能聊
- **飞书**：在飞书里搜索你的 Bot 名称
- **豆包**：在豆包 APP 里使用

## 五、常见问题

**Q: 直接对话 ChatGPT 和用智能体有什么区别？**

| | 直接对话大模型 | 智能体 |
|------|------|------|
| 比喻 | 问一位"无所不知但记性差"的万能博士 | 亲手打造的"专属专家员工" |
| 记忆 | 无 | 有知识库 + 变量持久化 |
| 能力 | 纯对话 | 可调用插件、搜网页、读文件 |
| 分发 | 无法分发 | 一键发布多平台 |

**Q: AI 机器人和智能体有什么区别？**

AI 机器人（Bot）是智能体（Agent）的子集。Bot 专注聊天回复，Agent 能完成复杂任务：规划+调用工具+执行+反馈。
